Create a new course

Before registering a new course in the autograder tool, you have to do one simple step on Github. Make a new organisation on their page.

Tip: Naming it after our template is often usefull. The name often follows the pattern uis-'couse code'-'semester time'.

Example: uis-dat320-fall2014

Get it for free

Private Github organisations and repositories is free for academic purpuses. However this has to be requested.

After creating a organisation on Github you can go to page linked below and fill out their form there.

When all of this is done you can proceed to pick the organisation.
