Autograder: Automated student feedback.

Autograder is a project created to give the students fast feedback on their lab work and prepre them on use of tools used in the industry. Autograder builds upon version control systems and continuous integration. When students upload code to their repositories, they can automatically build their code and get feedback from test supplied from teachers. when grading assignments, teachers can access this and have a valueble tool in the grading process.

Integrated with github: Manageing courses and students.

Within a course all students gets organized through an organization on GitHub. In this organization students gets access to their own repository. With this repository the students can store their code while working on it and hand in their code for review or grading by a teacher.

Continuous integration: instantanuous build and tests students code.

As code gets pushed to to github, an automatic build process, defined by the teacher, generate feedback on how well latest code preforms. When the build process is completed, students imidiatly gets access to this feedback on their personal course page. Tests defined by either teachers or students will be processed and tell students about their progress on the assignments.

Grading: Powerfull and easy.

When the due date is there teachers can access the test results and use this as a tool in the grading process. The teachers will straight away know about which parts of the code passes their tests and how much of the code is covered by the tests.
